# Anime bites: Um Livro de Receitas Inspirado em Cenas de Anime;)

## 📒 Descrição
"Anime Bites" é um livro de receitas inspirado em cenas de anime, trazendo pratos simples e fáceis de recriar diretamente da tela para o seu prato. Se você sempre quis fazer as comidas deliciosas que aparecem no seu anime favorito, este livro é para você! Cada receita foi criada para ser fácil de seguir, com ingredientes acessíveis e um toque de magia dos animes

## 🤖 Tecnologias Utilizadas
**GPT-3** para organizar o conteúdo e estruturar as receitas;
- Ferramentas de Design de IA para criar a capa do livro;
- Outras ferramentas de design para a diagramação do conteúdo do eBook.

## 🧐 Processo de Criação
A ideia para "Anime Bites" surgiu da minha paixão por animes e culinária. Usei o CHATGPT para ajudar a organizar e estruturar as receitas, e também para fornecer dicas e informações sobre como tornar as receitas mais acessíveis. A capa foi criada com o uso de IA, e o processo de diagramação e formatação foi feito com ferramentas de design digital para garantir um visual atraente.
